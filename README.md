# 📝 React Todo App

A modern Todo web application built with **React** and **Vite**.

---

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/react-todo-app.git
cd react-todo-app
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

Open:

```bash
http://localhost:5173
```
